## Telecom - customer churn
   
| Название проекта | Описание | Использованные библиотеки |
| :-------------------- | :--------------------- |:---------------------------:|
| Telecom | Оператору связи «Ниединогоразрыва.ком» необходимо прогнозировать отток клиентов. Если выяснится, что пользователь планирует уйти, ему будут предложены промокоды и специальные условия. Команда оператора собрала персональные данные о некоторых клиентах, информацию об их тарифах и договорах. Наша задача построить модель, которая поможет предсказать уход клиента. | *pandas sklearn seaborn*|
